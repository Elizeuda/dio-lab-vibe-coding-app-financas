# Projeto Final – Aplicativo de Organização de Finanças Pessoais

## Prompt Final (PRD)
### Contexto
O produto é um aplicativo de organização de finanças pessoais que funciona por meio de conversas em linguagem natural.  
Objetivo: simplificar o controle financeiro, eliminando formulários manuais e planilhas complexas, oferecendo uma experiência fluida, acessível e inclusiva, baseada em princípios de **Design Universal**.

### Problema
Os usuários desistem de controlar seus gastos porque os aplicativos atuais:
- Exigem muita entrada manual.
- Oferecem pouca personalização.
- Não são acessíveis para diferentes perfis de usuários.

O produto busca resolver isso com uma experiência conversacional, recomendações automáticas de economia e Design Universal, garantindo que pessoas com diferentes níveis de habilidade tecnológica ou necessidades específicas possam usar o aplicativo com facilidade.

### Público-Alvo
- Pessoas iniciantes no controle financeiro.  
- Usuários que desejam praticidade e simplicidade.  
- Pessoas que não têm familiaridade com planilhas ou sistemas complexos.  
- Usuários com diferentes necessidades de acessibilidade (ex.: visão reduzida, dificuldades motoras, baixa alfabetização digital).

### Funcionalidades-Chave
1. Registro de gastos via chat em linguagem natural.  
2. Classificação automática das transações.  
3. Definição e acompanhamento de metas financeiras.  
4. Agente Financeiro que fornece dicas de economia personalizadas.  
5. Relatórios simples e personalizados para acompanhamento.  
6. Acessibilidade integrada: interface clara, suporte a leitores de tela, linguagem simples e opções visuais adaptáveis.

### Entregável da IA
Plano de MVP (Produto Mínimo Viável) com:
- Principais telas (chat, metas, relatórios).  
- Recursos necessários (PNL, categorização automática, dashboards).  
- Esboço de validação inicial (testes com usuários iniciantes e com diferentes perfis de acessibilidade).  
- Linguagem acessível e educativa em português.

### Critérios de Sucesso
- Usuários conseguem registrar gastos sem esforço manual.  
- Metas financeiras são detalhadas e acompanhadas com clareza.  
- Relatórios são simples e úteis para iniciantes.  
- Feedback positivo sobre a experiência conversacional.  
- Usuários com diferentes necessidades relatam boa experiência de uso (Design Universal validado).

## Resumo do App
O aplicativo de Finanças Pessoais permite:
- Registrar gastos via chat em linguagem natural.  
- Classificar automaticamente transações.  
- Definir e acompanhar metas financeiras.  
- Receber dicas personalizadas de economia.  
- Visualizar relatórios simples e motivacionais.  
- Usar uma interface inclusiva e acessível, baseada em Design Universal.

## Reflexão sobre o Processo

### O que funcionou bem?
